# Contributing to what-new-pkg

Thank you for your interest in contributing.

## Development setup

```bash
git clone https://github.com/mayukh-dsc/what-new-pkg.git
cd what-new-pkg
npm ci
npm run build
npm test
```

## Pull requests

- Run `npm run typecheck`, `npm run lint`, and `npm test` before opening a PR.
- Keep changes focused and documented in [CHANGELOG.md](./CHANGELOG.md) when user-visible.
